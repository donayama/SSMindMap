# SSMindMap

ブラウザだけで動くシングルファイル・マインドマップアプリ。インストール不要。

🔗 **[https://donayama.github.io/SSMindMap/](https://donayama.github.io/SSMindMap/)**

---

## 主な機能

### レイアウト
| モード | 説明 |
|--------|------|
| 左右ツリー | デフォルト。ルートを中心に左右へ展開 |
| 上下ツリー | 上から下へ展開 |
| 右のみ / 左のみ / 下のみ / 上のみ | 一方向展開 |
| 放射状 | ノードを自由配置できるラジアルレイアウト |
| マンダラチャート | 3×3 グリッド × 9 ブロック（最大 8 テーマ × 8 アイデア） |

### ノード操作
- **追加** — `Tab`（子）/ `Enter`（兄弟）/ サイドバーボタン
- **編集** — ダブルクリック / `F2` / 文字キー（即時入力）
- **削除** — `Del`
- **コピー** — `Ctrl+D`
- **カット＆ペースト** — `Ctrl+X` → 貼り付け先を選んで `Ctrl+V`
- **ドラッグで親変更** — ノードをドラッグして別ノードへドロップ（ツリーモード）
- **折り畳み** — ノード右端の ＋/− ボタン

### テキスト装飾（ノードラベル内 Markdown）
| 記法 | 表示 |
|------|------|
| `**太字**` | **太字** |
| `*斜体*` | *斜体* |
| `__下線__` | <u>下線</u> |
| `~~取消線~~` | ~~取消線~~ |
| `https://...` | クリッカブルリンク |

### ノードスタイル
- **文字サイズ** — XS / S / M（デフォルト）/ L / XL
- **形状** — 角丸矩形 / 楕円 / ひし形
- **カラー** — 11 色プリセット（継承または個別設定）
- **雲グループ** — ノードとその子孫を雲形で囲む
- **絵文字** — ノードラベルに絵文字を挿入
- **メモ（吹き出し）** — ノードに 1 件のメモを吹き出し表示

### 関連線
ノード間に矢印付き曲線を引けます（関連モードボタンで有効化）。  
色 / 線種（実線・破線・点線・一点鎖線）/ 太さ / 矢印方向（→ / ↔ / なし）を個別設定可能。

### 検索・ナビゲーション
- **ノード検索** — サイドバー上部の検索欄（ヒットノードをハイライト）
- **ツリービュー** — 階層構造をサイドバーに表示、クリックで選択
- **ミニマップ** — 右下に全体俯瞰

### エクスポート
| 形式 | 説明 |
|------|------|
| JSON | 再読み込み可能な完全データ |
| YAML | 人が読みやすい形式 |
| Markdown | 箇条書き階層 |
| PNG | 画像として保存 |
| PDF | PDF として保存 |

### 自動保存
操作後 1.5 秒でブラウザのローカルストレージに自動保存。  
次回起動時に前回の作業を自動復元します。

### AI アイデア提案
サイドバーの「✨ AI アイデア提案」パネルでプロンプトを生成 →  
ChatGPT / Claude 等に貼り付け → 返答を貼り付けてノード自動追加。

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Tab` | 子ノード追加 |
| `Enter` | 兄弟ノード追加 |
| `Del` / `Backspace` | ノード削除（確認あり） |
| `F2` | 編集（既存テキスト全選択） |
| 文字キー | 選択中ノードの編集開始 |
| `←` / `→` | 親 / 子ノードへ移動 |
| `↑` / `↓` | 兄弟ノードをループ移動 |
| `F` | 全体表示 |
| `+` / `-` | ズームイン / アウト |
| `Esc` | 選択解除 / 編集キャンセル / モード終了 |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Y` / `Ctrl+Shift+Z` | やり直す |
| `Ctrl+D` | ノードコピー |
| `Ctrl+X` | カット |
| `Ctrl+V` | ペースト |
| `Ctrl+S` | JSON 保存 |
| `Ctrl+N` | 新規マップ |
| `Ctrl+O` | ファイルを開く |

---

## データ形式

JSON または YAML でエクスポート・インポートできます。  
`index.html` 1 ファイルで完結しており、ローカルで開くだけで動作します。
