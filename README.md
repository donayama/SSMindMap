# SSMindMap

ブラウザだけで動くシングルファイル・マインドマップアプリ。インストール不要。

🔗 **[https://donayama.github.io/SSMindMap/](https://donayama.github.io/SSMindMap/)**

---

## 主な機能

### レイアウト
| モード | 説明 |
|--------|------|
| 左右ツリー | デフォルト。ルートを中心に左右へ展開 |
| 上下ツリー | 上から下へ展開 |
| 右のみ / 左のみ / 下のみ / 上のみ | 一方向展開 |
| 放射状 | ノードを自由配置できるラジアルレイアウト |
| マンダラチャート | 3×3 グリッド × 9 ブロック（最大 8 テーマ × 8 アイデア） |

### ノード操作
- **追加** — `Tab`（子）/ `Enter`（兄弟）/ サイドバーボタン
- **編集** — ダブルクリック / `F2` / 文字キー（即時入力）
- **削除** — `Del`（設定で確認あり/なしを切替）
- **コピー** — `Ctrl+D`
- **カット＆ペースト** — `Ctrl+X` → 貼り付け先を選んで `Ctrl+V`
- **親変更** — `Ctrl` / `⌘` + ドラッグ、またはカット＆ペースト
- **自由移動** — 放射状レイアウトでは通常ドラッグで位置移動
- **折り畳み** — ノード右端の ＋/− ボタン
- **移動ガイダンス** — キャンバス左下に現在の移動方法を表示

### テキスト装飾（ノードラベル内 Markdown）
| 記法 | 表示 |
|------|------|
| `**太字**` | **太字** |
| `*斜体*` | *斜体* |
| `__下線__` | <u>下線</u> |
| `~~取消線~~` | ~~取消線~~ |
| `https://...` | クリッカブルリンク |
| `[[マップ名]]` | 保存済みマップへの内部リンク |

### ノードスタイル
- **文字サイズ** — XS / S / M（デフォルト）/ L / XL
- **形状** — 角丸矩形 / カプセル / 楕円 / ひし形 / 六角形 / 平行四辺形
- **外周線** — 太さ、実線 / 破線 / 点線 / 一点鎖線 / 二重線
- **カラー** — 11 色プリセット（継承または個別設定）
- **雲グループ** — ノードとその子孫を雲形で囲む
- **絵文字** — ノードラベルに絵文字を挿入
- **メモ（吹き出し）** — ノードに 1 件のメモを吹き出し表示（表示衝突を簡易回避）

### 関連線
ノード間に矢印付き曲線を引けます（関連モードボタンで有効化）。  
色 / 線種（実線・破線・点線・一点鎖線）/ 太さ / 矢印方向（→ / ↔ / なし）を個別設定可能。

### 検索・ナビゲーション
- **ノード検索** — サイドバー上部の検索欄（ヒットノードをハイライト）
- **ノードツリー** — 階層構造を独立ペインに表示、クリックで選択
- **ツリー独立開閉** — ツリー上の開閉はキャンバス上のノード開閉と独立
- **ミニマップ** — 右下に全体俯瞰

### UI と設定
- **ペイン分離** — ノードツリー、ノードプロパティ、操作、AI、設定を整理
- **表示位置** — ノードツリーとノードプロパティを左 / 右 / 非表示に切替
- **ヘッダ最適化** — よく使うノード操作をヘッダに配置し、ファイル操作はメニュー化
- **モーダル中の誤操作防止** — ダイアログ表示中はキャンバス移動やアプリショートカットを抑止
- **削除確認設定** — ノード削除・関連線削除の確認表示をON/OFF可能

### エクスポート
| 形式 | 説明 |
|------|------|
| JSON | 再読み込み可能な完全データ |
| YAML | 人が読みやすい形式 |
| Markdown | 箇条書き階層 |
| PNG | 画像として保存 |
| PDF | PDF として保存 |

### 複数マップと自動保存
- **ブラウザ内マップ一覧** — ヘッダの「マップ」セレクトで保存済みマップを切替
- **空白マップ追加** — ヘッダまたは設定パネルから新しいマップを追加
- **継続保存** — 同じマップIDへ自動保存し、次回起動時に復元
- **ファイル保存** — JSON / YAML / Markdown / PNG / PDF として明示保存

### AI アイデア提案
サイドバーの「✨ AI アイデア提案」パネルでプロンプトを生成 →  
ChatGPT / Claude 等に貼り付け → 返答を貼り付けてノード自動追加。

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Tab` | 子ノード追加 |
| `Enter` | 兄弟ノード追加 |
| `Del` / `Backspace` | ノード削除（確認あり） |
| `F2` | 編集（既存テキスト全選択） |
| 文字キー | 選択中ノードの編集開始 |
| `←` / `→` | 親 / 子ノードへ移動 |
| `↑` / `↓` | 兄弟ノードをループ移動 |
| `F` | 全体表示 |
| `+` / `-` | ズームイン / アウト |
| `Esc` | 選択解除 / 編集キャンセル / モード終了 |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Y` / `Ctrl+Shift+Z` | やり直す |
| `Ctrl+D` | ノードコピー |
| `Ctrl+X` | カット |
| `Ctrl+V` | ペースト |
| `Ctrl` / `⌘` + ドラッグ | 別の親ノードへ移動 |
| `Ctrl+S` | JSON 保存 |
| `Ctrl+N` | 新規マップ |
| `Ctrl+O` | ファイルを開く |

## データ形式

JSON または YAML でエクスポート・インポートできます。  
`index.html` 1 ファイルで完結しており、ローカルで開くだけで動作します。

## セルフテスト

UI/UX の主要フローは Playwright ベースのセルフテストで検証できます。

```bash
node scripts/e2e-self-test.mjs
```

主な検証範囲:
- ペイン表示位置、設定保存、削除確認ON/OFF
- ノード追加、編集、削除、コピー、カット、貼り付け
- 選択ハイライト、ノードツリー独立開閉、検索
- `Ctrl`+ドラッグ移動、関連線作成/編集/削除
- AI回答解析、メモ吹き出し、Undo/Redo
- 複数マップ切替、内部マップリンク
- JSON / Markdown / PNG / YAML / PDF 出力、JSON読込
- 狭幅ビューポートでの主要UI表示
