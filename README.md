# SSMindMap

ブラウザだけで動くマインドマップアプリ。インストール不要。

🔗 **[https://donayama.github.io/SSMindMap/](https://donayama.github.io/SSMindMap/)**

## 機能

- **ラジアルレイアウト**の自動配置
- **JSON / YAML** でのローカルファイル保存・読み込み
- **AI アイデア提案** — プロンプト生成 → ChatGPT / Claude 等に貼り付け → 回答を解析してノード追加
- **関連線** — ノード間に矢印付き破線を引く
- **雲グループ** — ノードとその子孫を雲形でグループ化
- ダーク / ライトテーマ
- キーボードショートカット

## キーボードショートカット

| キー | 動作 |
|------|------|
| `←` / `→` | 親 / 子ノードへ移動 |
| `↑` / `↓` | 兄弟ノードをループ移動 |
| 文字キー | 選択中ノードの編集開始 |
| `F2` | 編集（既存テキスト全選択） |
| `Tab` | 子ノード追加 |
| `Enter` | 兄弟ノード追加 |
| `Del` | ノード削除 |
| `F` | 全体表示 |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+S` | JSON保存 |
| `Esc` | 選択解除 / キャンセル |

## データ形式

JSON または YAML でエクスポート・インポートできます。
